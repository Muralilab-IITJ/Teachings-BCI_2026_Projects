# BCI 2026 Portfolio Website

Modern portfolio site built with **React + Vite + Tailwind CSS**.

## Run locally

1. Open terminal in this folder:
   `E:\Websites\BCI_2026\portfolio`
2. Install dependencies:
   `npm install`
3. Start dev server:
   `npm run dev`
4. Build production bundle:
   `npm run build`

## What this site includes

- Hero section with BCI portfolio overview
- Team project cards (A to G)
- Team member names for each project
- Direct links for code, method, and video/demo
- Responsive modern UI with gradient/glass style
